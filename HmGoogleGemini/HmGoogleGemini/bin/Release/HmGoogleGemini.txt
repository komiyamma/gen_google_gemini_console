
## ECMAScript 2022 の新機能

ECMAScript 2022 は、Nullish 병합 연산자 (??)、論理 OR 割り当て演算子 (||=)、論理 AND 割り当て演算子 (&&=) などのいくつかの新機能を導入しました。

### Nullish 병합 연산자 (??)

Nullish 병합 연산자 (??) は、左辺が `null` または `undefined` でない限り、左辺を返します。そうでない場合、右辺を返します。これは、条件付き演算子に似ていますが、左辺が `null` または `undefined` の場合にデフォルト値を指定できるという点で異なります。

```javascript
const name = user?.name ?? "デフォルト名";
```

この例では、`user.name` が `null` または `undefined` でない場合、`name` に `user.name` の値が代入されます。そうでない場合、`name` に "デフォルト名" が代入されます。


### 論理 OR 割り当て演算子 (||=)

論理 OR 割り当て演算子 (||=) は、左辺が真の場合には右辺を評価せず、左辺が偽の場合にのみ右辺を評価します。値はその後、左辺に割り当てられます。

```javascript
let x = 0;
x ||= 10;
console.log(x); // 10 を出力
```

この例では、`x` が真 (0 ではない) であるため、`10` は評価されず、`x` には `0` が割り当てられたままになります。


### 論理 AND 割り当て演算子 (&&=)

論理 AND 割り当て演算子 (&&=) は、左辺が偽の場合には右辺を評価せず、左辺が真の場合にのみ右辺を評価します。値はその後、左辺に割り当てられます。

```javascript
let x = 10;
x &&= 5;
console.log(x); // 5 を出力
```

この例では、`x` は真 (0 ではない) であるため、`5` が評価され、`x` には `5` が割り当てられます。
